# BOSS Helper

BOSS Helper 是一个 Tampermonkey 油猴脚本，用于在 BOSS 直聘网页职位列表中，按可配置频率自动点击 `立即沟通`，并在发送成功弹窗中点击 `留在此页`，继续处理下一个职位。

## 功能

- 在 BOSS 页面右下角显示控制面板。
- 支持开始、暂停、停止、重置今日统计。
- 默认每 `60` 秒处理一次。
- 默认每天最多成功沟通 `150` 个。
- 支持修改间隔秒数和每日上限。
- 支持配置必须包含关键词和排除关键词。
- 默认只投测试/测开相关岗位，默认排除销售、客服、顾问等岗位。
- 自动遍历左侧职位列表。
- 自动点击右侧详情里的 `立即沟通`。
- 发送成功后自动点击 `留在此页`。
- 跳过 `继续沟通`、`已沟通`、`去App`、扫码等状态。
- 当前列表处理完后自动下拉加载更多职位。
- 遇到验证码、登录、安全验证、访问频繁等提示会停止。

## 安装

1. 浏览器安装 Tampermonkey。
2. 打开 Tampermonkey 控制面板。
3. 新建脚本。
4. 将 `boss-helper.user.js` 的完整内容粘贴进去。
5. 保存脚本。
6. 打开 BOSS 直聘职位列表页面。

脚本文件路径：

```text
/boss-helper/boss-helper.user.js
```

## 使用

1. 先在 BOSS 页面手动完成登录。
2. 手动设置城市、职位、行业、薪资、经验等筛选条件。
3. 确认页面左侧出现职位列表，右侧出现职位详情。
4. 在右下角 BOSS Helper 面板中确认配置：
   - 间隔秒数：默认 `60`
   - 每日上限：默认 `150`
   - 必须包含：默认 `测试开发,测试工程师,软件测试,自动化测试,测开,QA,质量`
   - 排除关键词：默认 `网络销售,电话销售,销售,客服,社群销售,电销,地推,招商,顾问`
5. 点击 `开始`。
6. 需要中断时点击 `暂停` 或 `停止`。

## 配置

| 配置 | 默认值 | 说明 |
| --- | --- | --- |
| 间隔秒数 | `60` | 每次成功或跳过后等待多久再继续 |
| 每日上限 | `150` | 每天最多成功点击 `立即沟通` 的次数 |
| 必须包含 | `测试开发,测试工程师,软件测试,自动化测试,测开,QA,质量` | 职位卡片文本必须命中至少一个关键词；留空表示不启用必须包含过滤 |
| 排除关键词 | `网络销售,电话销售,销售,客服,社群销售,电销,地推,招商,顾问` | 职位卡片文本命中任一关键词就跳过 |

配置保存在 Tampermonkey 本地存储中。脚本不会保存聊天内容、cookie、token、手机号、招聘者信息或页面 HTML。

关键词可以用逗号、中文逗号、空格或换行分隔。例如：

```text
测试开发, 自动化测试, 测试工程师
```

过滤分两步执行：

1. 点开职位前，只看左侧卡片是否命中排除关键词，比如 `网络销售`、`电话销售`。
2. 点开职位后，再用左侧卡片文本加右侧详情文本一起判断是否命中“必须包含”。

这样可以避免左侧列表文本不完整时，把 `软件测试工程师` 误判为不匹配。

## 执行逻辑

```text
读取左侧职位卡片
→ 先检查排除关键词
→ 点开职位
→ 等待右侧详情加载
→ 检查必须包含关键词
→ 查找“立即沟通”
→ 点击“立即沟通”
→ 等待“已向BOSS发送消息”弹窗
→ 点击“留在此页”
→ 成功计数 +1
→ 等待配置间隔
→ 继续下一个职位
```

如果按钮是 `继续沟通`、`已沟通`、`去App` 或扫码提示，脚本会跳过当前职位，不增加成功计数。

## 安全边界

这个脚本只做页面内辅助点击，不做以下事情：

- 不绕过验证码。
- 不自动登录。
- 不绕过安全验证。
- 不隐藏自动化行为。
- 不调用 BOSS 私有接口。
- 不抓取或导出招聘者个人信息。
- 不修改打招呼内容。
- 不发送任何外部网络请求。

如果页面出现验证码、登录失效、安全验证、访问频繁或未知弹窗，脚本会停止并在面板日志中提示原因。

## 测试

运行核心逻辑测试：

```bash
node /boss-helper/tests/boss-helper-core.test.js
```

手工测试建议：

1. 先把每日上限改成 `1`。
2. 点击 `开始`。
3. 观察是否只沟通一个职位。
4. 确认成功弹窗出现后点击的是 `留在此页`。
5. 确认成功数变成 `1` 后停止。

## 已知限制

- BOSS 页面 DOM 结构变化时，选择器可能需要调整。
- 页面如果强制跳 App 或二维码沟通，脚本会跳过。
- 如果浏览器标签页休眠，计时可能延迟。
- 如果页面出现未知弹窗，脚本会保守停止。
